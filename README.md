# English Trainer Bot

Telegram-бот для изучения английских слов. Проект помогает пользователям учить и повторять слова с помощью интерактивных карточек и систематического повторения.

## Описание проекта

Проект представляет собой Telegram-бота, реализованный на Python с использованием библиотеки `pyTelegramBotAPI`. Бот взаимодействует с пользователем, предлагая изучать новые слова и повторять уже выученные. Данные о словах хранятся в локальной базе данных, структура которой определяется через SQLAlchemy.

База слов была получена из стороннего проекта: [https://github.com/OttomaniaKazan/parser_langwich_ru-self_learn-](https://github.com/OttomaniaKazan/parser_langwich_ru-self_learn-), который парсит слова с сайта Langwich. Слова хранятся в формате JSON и загружаются в базу данных при инициализации.

## Установка и запуск

### 1. Установка зависимостей

Убедитесь, что у вас установлен Python 3.8 или выше. Затем установите зависимости из файла `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 2. Настройка окружения

Создайте файл `.env` на основе примера `.env_exmpl`, указав ваш токен от BotFather:

```env
TOKEN=ваш_токен_здесь
```

### 3. Инициализация базы данных

Перед первым запуском бота необходимо инициализировать базу данных. Для этого выполните скрипт:

```bash
python db_first_start.py
```

Этот скрипт создаст необходимые таблицы в базе данных и загрузит словарь из `vocabulary.json`.

> **Важно:** Скрипт `db_first_start.py` следует запускать только при первом старте или при необходимости пересоздать базу данных. Он вызывает `create_tables()`, который определён в `database/db_init.py`.

### 4. Запуск бота

После инициализации базы данных запустите бота:

```bash
python main.py
```

Бот начнёт polling-режим и будет отвечать на сообщения пользователей.

## Использование

После запуска бота вы можете начать с диалога с ним в Telegram. Доступные команды:

- `/start` — начать работу с ботом

Бот отслеживает прогресс пользователя и использует алгоритм интервальных повторений для эффективного запоминания.

## Структура проекта

- `bot/` — логика бота, обработчики команд и состояния
- `database/` — работа с базой данных, модели, инициализация
- `decorators/` — пользовательские декораторы
- `main.py` — точка входа
- `db_first_start.py` — инициализация базы данных
- `requirements.txt` — зависимости
